Live Stream of What I am Playing on Spotify

# Setup

1. Connect Spotify to Last.fm
2. Get an Last.fm API Key
3. set up in `config/lastfm.js` and `config/index.js`
4. `npm install` or `yarn`
5. `yarn start`
